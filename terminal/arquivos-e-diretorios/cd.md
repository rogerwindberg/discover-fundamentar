# cd

Descrição: cd (Change Directory) altera diretórios

cd . -> Diretório atual

cd .. -> Voltar diretórios

cd - -> Voltar ao mesmo diretório onde estava

cd "Diretório com espaços" -> Navega até diretórios que contenham espaços no nome

cd Diretório\ com\ espaços -> Navega até diretórios que contenham espaços no nome
